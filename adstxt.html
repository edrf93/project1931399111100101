Ads.txt Implementation Guide

Secure your ad inventory and maximize revenue by implementing the simple Ads.txt file on your root domain.

What is Ads.txt?

Ads.txt (Authorized Digital Sellers) is an IAB initiative designed to increase transparency in programmatic advertising. By listing authorized sellers, you prevent unauthorized parties from selling your inventory, which protects your brand and increases buyer confidence.

How to Implement AdRyx's Ads.txt Line

  1. Create a plain text file named ads.txt.
  2. Copy and paste the authorized line(s) provided by AdRyx below into this file.
  3. Upload the file to the root directory of your website (e.g., https://yourdomain.com/ads.txt).
  4. If you have existing lines, simply add our line to the end of the file.

Your AdRyx Line:

adryx.io, [PUBLISHER_ID], DIRECT, [CERTIFICATION_AUTHORITY_ID]

Important: Please replace [PUBLISHER_ID] with the unique ID provided to you by your AdRyx Yield Manager after application approval.
